🏠 House Price Analysis

Overview
This project analyzes housing data to identify key factors influencing house prices using Python and data analysis techniques.

---

Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn

---

Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature impact analysis (area, bedrooms, bathrooms)
- Visualization of price trends

---

🔍 Key Insights
- Larger area strongly increases house price
- Number of bedrooms and bathrooms impacts pricing
- Amenities like parking and air conditioning increase value

---

How to Run
1. Open the notebook:
   `Housing Price.ipynb`
2. Run all cells
